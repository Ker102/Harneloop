# Harneloop v0.0.2 - Public Alpha

This release brings the public alpha tag up to the current agent-operated harness lifecycle.

## Highlights

- Globally addressable harness units and unit-scoped context recovery.
- Adaptive intake that records assumptions and requests material missing context.
- Explicit attempt conclusions so runs do not stop before evidence is assessed.
- Parallel candidate batches with impact-aware validation and rebase rules.
- Stronger immutable-run and evidence-integrity checks.
- Agent onboarding guidance for environment mapping, capability gaps, and reuse of suitable existing tools.
- Updated documentation, public website metadata, and package metadata.

## Install

Until the first PyPI publication completes:

```bash
uv tool install git+https://github.com/Ker102/Harneloop.git@v0.0.2
harneloop doctor
```

Harneloop is still an alpha. Commands and managed file formats may change before a stable release.
