# Core MVP Implementation Plan

> **For Agent:** Use executing-plans skill to implement this plan task-by-task.

**Goal:** Build the first task-family-neutral Harneloop lifecycle engine.

**Architecture:** A Python CLI manipulates portable harness-unit directories. The engine stores human-readable YAML and Markdown control files, creates candidate patch workspaces, validates protected boundaries, promotes candidates into restorable snapshots, supports rollback, creates thin packages, and records wait/stop/resume state.

**Tech Stack:** Python 3.11+, PyYAML, standard-library `argparse`, `tarfile`, `hashlib`, and `unittest`.
